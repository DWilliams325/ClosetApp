Quick Start Tutorial

1) Add items

  1. Pick a Category and (optional) Color.
  2. Tap Upload photo (or drag & drop on desktop).
  3. Press Add Item.

Tip: The app remembers your last category to speed things up.

2) Speed Add (camera)

  1. Tap Speed Add (Camera).
  2. Choose a category at the top.
  3. Tap the big shutter to capture; thumbnails appear below.
  4. Tap Save All to add them at once.

3) Add via image URL

  1. Copy a direct link ending in .jpg, .png, or .webp.
  2. Paste it into Or paste an image URL ‚Üí Use URL.
  3. Press Add Item.

iPhone: long-press the image in Safari ‚Üí Open Image in New Tab ‚Üí copy the address bar.

4) Categories & colors

  • Use the chips under Virtual Closet to filter; tap All to reset.
  • Shoe types supported: Heels, Sneakers, Boots, Dress Shoes, Sandals, Flats.

5) Build an outfit

  1. Tap Make Outfit.
  2. Filter by category/color if you like.
  3. Tap items in Your Closet to select/unselect.
  4. Selected pieces appear under Tops, Bottoms, and Shoes.
  5. (Optional) Name it, then Save Outfit.

6) Edit / delete items

On any card, use Edit to change category, color, or photo; use Delete to remove.

7) Share / Import

Generate a link that encodes your items so someone else can preview, merge, or replace their closet.
